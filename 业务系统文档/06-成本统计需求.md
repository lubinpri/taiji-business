# 成本统计模块 - 需求规格说明书

## 一、业务流程

```
项目管理 → 工时确认 → 人力成本自动计算
     ↓
  其他成本（采购/外包/差旅/其他）
     ↓
  成本统计 → 利润核算 → 成本分析
```

---

## 二、功能清单

### 2.1 项目成本统计 [P0]

| 功能 | 说明 |
|:---|:---|
| 项目成本汇总 | 按项目查看总成本 |
| 成本明细 | 按类型查看成本明细 |
| 预算对比 | 预算成本 vs 实际成本 |
| 利润计算 | 收入 - 成本 - 税金 = 利润 |
| 毛利率 | 利润 / 收入 |

### 2.2 人力成本统计 [P0]

| 功能 | 说明 |
|:---|:---|

### 2.3 部门成本核算 [P0]

## 七、核心计算逻辑

### 7.1 项目利润（含税金）

```
项目利润 = 合同收入 - 人力成本 - 采购成本 - 外包成本 - 差旅成本 - 其他成本 - 税金

其中：
税金 = 合同收入 × 税率（如6%）
```

### 7.2 人力成本自动计算

```
人力成本 = Σ (每人实际人天 × 该人人天单价)
```

### 7.3 部门成本核算

```
部门总成本 = 人力成本 + 房租分摊 + 办公成本 + 交通成本 + 招待成本 + 其他成本

部门利润 = 部门项目收入合计 - 部门总成本
部门毛利率 = 部门利润 / 部门收入 × 100%
```

### 7.4 税金分摊

```
项目税金 = 项目收入 × 税率
（税金通常在项目回款时计算）
```
