# 投标管理模块 - 需求规格说明书

## 一、业务流程

```
发现商机 → 决定投标 → 投标准备 → 成本预算 → 投标审批 → 中标/失标
                                                    ↓
                                             签约转合同/项目
```

---

## 二、功能清单

### 2.1 投标基础管理 [P0]

| 功能 | 说明 |
|:---|:---|
| 创建投标 | **必须关联商机**，无商机不可创建 |
| 编辑投标 | 修改投标信息 |
| 投标列表 | 列表展示，支持筛选 |
| 投标详情 | 查看完整投标信息 |

### 2.2 成本预算 [P0]

| 功能 | 说明 |
|:---|:---|
| 人力成本预算 | 估算参与人员、工时、人天单价 |
| 采购成本预算 | 估算硬件/软件采购 |
| 外包成本预算 | 估算第三方服务 |
| 差旅成本预算 | 估算出差费用 |
| 其他成本预算 | 其他支出估算 |
| 成本汇总 | 自动计算总成本和利润 |
| 成本变更 | 实施阶段可修改预算 |

### 2.3 投标文件 [P1]

| 功能 | 说明 |
|:---|:---|
| 上传投标文件 | 支持Word/PDF |
| 标书版本 | 保留历史版本 |

### 2.4 投标审批 [P1]

| 功能 | 说明 |
|:---|:---|
| 提交审批 | 提交投标申请 |
| 审批流 | 参考成本+方案进行审批 |
| 审批记录 | 完整审批历史 |

### 2.5 投标结果 [P0]

| 功能 | 说明 |
|:---|:---|
| 中标登记 | 记录中标信息 |
| 失标登记 | 记录失标原因 |
| 转合同/项目 | 中标后自动/手动转合同管理 |

## 三、字段设计

### 3.1 投标信息

| 字段 | 类型 | 必填 | 说明 |
|:---|:---|:---:|:---|
| 投标名称 | 文本(100) | ✅ | 投标项目名称 |
| 关联商机 | 关联 | ✅ | 关联的商机 |
| 客户名称 | 文本 | ✅ | 甲方公司 |
| 投标金额 | 数值 | ✅ | 报价金额 |
| 投标截止日期 | 日期 | ✅ | 投标截止时间 |
| 项目类型 | 枚举 | - | 软件开发/集成/服务/运维/其他 |
| 项目经理 | 用户 | - | 拟派项目经理 |
| 销售经理 | 用户 | ✅ | 负责销售 |
| 投标状态 | 枚举 | - | 准备中/已提交/已开标 |

## 四、权限设计

### 4.1 角色职责

| 角色 | 职责 |
|:---|:---|
| 销售经理 | 创建投标、填写报价、提交审批 |
| 销售 | 创建投标、填写报价（自己创建的） |
| 项目经理 | 填写成本预算（人力/采购/外包等） |
| 商务 | 成本复核、查看统计、跟进风险 |
| 管理员 | 全部权限 |

| 操作 | 管理员 | 销售经理 | 销售 | 项目经理 | 商务 |

| 创建投标 | ✅ | ✅ | ✅ | ❌ | ❌ |
| 查看自己投标 | ✅ | ✅ | ✅ | 仅自己 | ✅ |
| 查看全部投标 | ✅ | ✅ | ❌ | ❌ | ✅ |
| 填写报价 | ✅ | ✅ | ✅ | ❌ | ❌ |
| 填写成本预算 | ✅ | ❌ | ❌ | ✅ | ❌ |
| 修改采购成本 | ✅ | ❌ | ✅（投标准备）| ✅（实施阶段）|
| 提交审批 | ✅ | ✅ | ❌ | ❌ |
| 审批投标 | ✅ | ✅ | ❌ | ❌ |
| 查看统计 | ✅ | ✅ | 仅自己 | ✅ |
| 成本复核 | ✅ | ❌ | ❌ | ✅ |

## 六、与其他模块的关系

```
商机 → 投标 → 成本预算 → 投标审批
                          ↓
                      中标 → 合同管理
                          ↓
                      项目管理（成本确认）
```
