## 🔗 表关系图

```
users (用户)
  ↑
  ├─ owner_id → opportunities (商机)
  ├─ owner_id → contracts (合同)
  └─ manager_id → projects (项目)

opportunities (商机)
  └─ id → contracts.opportunity_id (合同)

contracts (合同)
  └─ id → projects.contract_id (项目)

projects (项目)
  └─ id → transactions.project_id (收支)
```

---

## 📝 索引建议

```sql
-- 商机按状态查询
CREATE INDEX idx_opportunities_stage ON opportunities(stage);

-- 合同按客户查询
CREATE INDEX idx_contracts_customer ON contracts(customer_name);

-- 项目按状态查询
CREATE INDEX idx_projects_status ON projects(status);

-- 收支按项目查询
CREATE INDEX idx_transactions_project ON transactions(project_id);
```
