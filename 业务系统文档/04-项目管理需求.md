# 项目管理模块（重制版）- 需求规格说明书

## 一、业务流程

```
合同签订 → 项目立项 → 项目执行 → 项目验收 → 项目结项 → 成本核算
     ↑                                            ↓
投标成本预算 ←←←←← 成本确认/变更 ←←←←← 工时管理 ←←← 工时计划
```

---

## 二、功能模块

### 2.1 项目基础管理 [P0]

| 功能 | 说明 |
|:---|:---|
| 创建项目 | 关联合同，继承投标成本预算 |
| 编辑项目 | 修改项目信息 |
| 项目列表 | 列表展示，支持筛选/排序 |
| 项目详情 | 查看完整项目信息 |

### 2.2 成本管理 [P0]

#### 2.2.1 成本确认/变更

| 功能 | 说明 |
|:---|:---|
| 查看预算 | 从投标继承成本预算 |
| 成本调整 | 商务根据实际情况修改预算 |
| 成本确认 | 商务确认后成本生效 |
| 成本变更记录 | 记录所有变更，可追溯 |

#### 2.2.2 成本类型

| 类型 | 填报人 | 确认人 | 说明 |
|:---|:---|:---|:---|
| 人力成本 | 项目经理 | 自动计算 | 根据工时确认计算 |
| 采购成本 | 商务 | 商务确认 | 实际采购金额 |
| 外包成本 | 商务 | 商务确认 | 实际外包费用 |
| 差旅成本 | 项目经理/商务 | 商务确认 | 实际差旅费用 |
| 其他成本 | 项目经理/商务 | 商务确认 | 其他实际支出 |

### 2.3 工时管理 [P0]

#### 2.3.1 工时计划

| 功能 | 说明 |
|:---|:---|
| 维护参与人员 | 项目经理指定交付人员 |
| 计划人天 | 为每人计划投入人天 |
| 计划调整 | 可根据实际情况调整 |

#### 2.3.2 工时确认

| 功能 | 说明 |
|:---|:---|
| 周工时填报 | 每周确认实际投入人天 |
| 工时历史 | 查看历史工时记录 |
| 工时统计 | 自动汇总人力成本 |

### 2.4 里程碑管理 [P0]

| 功能 | 说明 |
|:---|:---|
| 里程碑模板 | 按项目类型预设里程碑 |
| 自定义里程碑 | 支持添加自定义里程碑 |
| 里程碑完成 | 标记完成状态 |

### 2.5 项目进度 [P0]

| 功能 | 说明 |
|:---|:---|
| 进度更新 | 每周更新项目进度 |
| 超期提醒 | 进度滞后自动提醒 |
| 风险标记 | 标记项目风险等级 |

### 2.6 项目结项 [P0]

| 功能 | 说明 |
|:---|:---|
| 提交结项 | 项目经理提交结项申请 |
| 上传验收文件 | PDF格式验收确认 |
| 结项附件 | Word/PDF/MP3文档 |
| 项目结项 | 项目经理确认完成 |

## 三、字段设计

### 3.1 项目信息

| 字段 | 类型 | 必填 | 说明 |
|:---|:---|:---:|:---|
| 项目名称 | 文本(100) | ✅ | 项目标题 |
| 项目编号 | 文本 | ✅ | 自动生成 |
| 合同 | 关联 | ✅ | 关联合同 |
| 客户名称 | 文本 | ✅ | 甲方公司 |
| 项目类型 | 枚举 | - | 软件开发/集成/服务/运维 |
| 项目经理 | 用户 | ✅ | 负责人 |
| 销售经理 | 用户 | - | 签约销售 |

### 3.2 成本相关

#### 3.2.1 成本预算（从投标继承）

| 字段 | 类型 | 说明 |
|:---|:---|:---|
| 预算人力成本 | 数值 | 投标阶段估算 |
| 预算采购成本 | 数值 | 投标阶段估算 |
| 预算外包成本 | 数值 | 投标阶段估算 |
| 预算差旅成本 | 数值 | 投标阶段估算 |
| 预算其他成本 | 数值 | 投标阶段估算 |
| 预算总成本 | 数值 | 自动汇总 |
| 预算毛利率 | 数值 | 自动计算 |

#### 3.2.2 成本确认

| 字段 | 类型 | 说明 |
|:---|:---|:---|
| 确认人力成本 | 数值 | 根据工时确认计算 |
| 确认采购成本 | 数值 | 商务确认 |
| 确认外包成本 | 数值 | 商务确认 |
| 确认差旅成本 | 数值 | 商务确认 |
| 确认其他成本 | 数值 | 商务确认 |
| 确认总成本 | 数值 | 自动汇总 |
| 成本确认状态 | 枚举 | 待确认/已确认 |
| 成本确认人 | 用户 | 商务确认 |
| 成本确认时间 | 日期 | 确认时间 |

#### 3.2.3 成本变更

| 字段 | 类型 | 说明 |
|:---|:---|:---|
| 变更内容 | 文本 | 变更说明 |
| 变更前金额 | 数值 | 变更前 |
| 变更后金额 | 数值 | 变更后 |
| 变更原因 | 文本 | 变更原因 |
| 变更人 | 用户 | 变更人 |
| 变更时间 | 日期 | 变更时间 |

### 3.3 工时相关

#### 3.3.1 参与人员

| 字段 | 类型 | 说明 |
|:---|:---|:---|
| 成员 | 用户 | 交付人员 |
| 角色 | 文本 | 如：开发、测试 |
| 计划人天 | 数值 | 计划投入 |
| 已投入人天 | 数值 | 累计已确认 |
| 剩余人天 | 数值 | 计划 - 已投入 |

#### 3.3.2 工时确认

| 字段 | 类型 | 说明 |
|:---|:---|:---|
| 周期 | 枚举 | 第X周 |
| 年份 | 数值 | 年份 |
| 成员 | 用户 | 人员 |
| 实际人天 | 数值 | 本周实际投入 |
| 工时说明 | 文本 | 工作内容说明 |
| 确认时间 | 日期 | 填报时间 |

### 3.4 进度相关

| 字段 | 类型 | 说明 |
|:---|:---|:---|
| 当前进度 | 数值(0-100) | 完成百分比 |
| 风险等级 | 枚举 | 正常/黄色预警/红色预警 |
| 最近更新 | 日期时间 | 最后更新时间 |

### 3.5 里程碑

| 字段 | 类型 | 说明 |
|:---|:---|:---|
| 里程碑名称 | 文本 | 名称 |
| 计划完成日期 | 日期 | 计划 |
| 实际完成日期 | 日期 | 实际 |
| 完成状态 | 枚举 | 未开始/进行中/已完成 |

## 五、风险规则

### 5.1 进度风险

| 风险等级 | 条件 |
|:---|:---|
| 🟢 正常 | 进度≥计划进度 |
| 🟡 黄色预警 | 进度滞后<15% |
| 🔴 红色预警 | 进度滞后≥15% |

### 5.2 成本风险

| 风险等级 | 条件 |
|:---|:---|
| 🟢 正常 | 实际成本≤预算成本 |
| 🟡 黄色预警 | 实际成本>预算成本<110% |
| 🔴 红色预警 | 实际成本≥预算成本110% |

## 六、页面结构

### 6.1 项目详情 - 成本

```
┌─────────────────────────────────────────────────────────┐
│  项目详情                          [成本确认] [编辑]   │
├─────────────────────────────────────────────────────────┤
│  成本预算（投标阶段）                                    │
│  ├─ 人力: ¥23,000  采购: ¥150,000 外包: ¥30,000     │
│  └─ 总成本: ¥218,000  毛利率: 56.4%                   │
│                                                         │
│  成本确认（实施阶段）                                    │
│  ├─ 人力: ¥28,000  采购: ¥145,000 外包: ¥25,000     │
│  └─ 总成本: ¥210,000  毛利率: 58%      [待商务确认]  │
│                                                         │
│  成本变更记录:                                          │
│  ├─ 02-01: 采购成本 ¥150,000→¥145,000 (节省5千)    │
│  └─ 02-15: 人力成本 ¥23,000→¥28,000 (增加5千人天)   │
└─────────────────────────────────────────────────────────┘
```

### 6.2 项目详情 - 工时

```
┌─────────────────────────────────────────────────────────┐
│  工时管理                          [添加成员]          │
├─────────────────────────────────────────────────────────┤
│  参与人员:                                              │
│  ├─ 张三(开发): 计划20人天 / 已投入15人天 / 剩5      │
│  ├─ 李四(测试): 计划10人天 / 已投入8人天 / 剩2      │
│  └─ 王五(设计): 计划8人天 / 已投入2人天 / 剩6       │
├─────────────────────────────────────────────────────────┤
│  本周工时确认 (第8周):                                 │
│  ├─ 张三: 5人天 ✓ 已确认                            │
│  ├─ 李四: 3人天 ✓ 已确认                            │
│  └─ 王五: 2人天 ✓ 已确认                            │
│                                                  [确认本周] │
├─────────────────────────────────────────────────────────┤
│  历史工时:                                              │
│  ├─ 第7周: 张三3 + 李四3 + 王五2 = 8人天           │
│  ├─ 第6周: 张三4 + 李四3 + 王五0 = 7人天           │
│  └─ ...                                               │
└─────────────────────────────────────────────────────────┘
```

## 七、流程图

```
                    投标阶段
                      ↓
              成本预算（项目经理）
                      ↓
                    中标
                      ↓
        ┌───────────┴───────────┐
        ↓                       ↓
   合同管理                项目管理
        │                       ↓
        │              成本确认/变更（商务）
        │                       ↓
        │                 工时计划（项目经理）
        │                       ↓
        │                 工时确认（每周）
        │                       ↓
        └──────→ 成本统计 ←────┘
                      ↓
                   成本核算 → 成本结项
```
