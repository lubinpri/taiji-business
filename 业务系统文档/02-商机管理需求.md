# 商机管理模块 - 需求规格说明书

## 一、业务流程设计

### 1.1 销售漏斗

```
潜在客户 → 需求确认 → 方案报价 → 商务谈判 → 已签约
   (100%)     (60%)       (40%)       (25%)
    ↓                                        ↓
           已失效（流失/放弃）
```

### 1.2 阶段定义

| 阶段 | 名称 | 说明 | 预计转化率 |
|:---:|:---|:---|:---:|
| 1 | 潜在客户 | 刚获取的客户信息 | 100% |
| 2 | 需求确认 | 已确认客户有明确需求 | 60% |
| 3 | 方案报价 | 已提供解决方案/报价 | 40% |
| 4 | 商务谈判 | 洽谈合同条款 | 25% |
| 5 | 已签约 | 成功签约 | - |
| 6 | 已失效 | 客户流失/放弃 | - |

## 二、功能清单

### 2.1 商机基础管理 [P0]

| 功能 | 说明 |
|:---|:---|
| 创建商机 | 新增商机信息（客户名、联系人、金额等） |
| 编辑商机 | 修改商机信息 |
| 删除商机 | 软删除，可恢复 |
| 商机列表 | 查看所有商机，支持筛选/排序 |
| 商机详情 | 查看完整商机信息及跟进记录 |
| 商机转移 | 将商机转给其他销售 |
| 大金额确认 | 商机金额超过阈值需上级确认 |

### 2.2 报价管理 [P0]

| 功能 | 说明 |
|:---|:---|
| 创建报价 | 记录报价金额、报价说明 |
| 报价附件 | 支持上传报价文件（PDF/Word/Excel） |
| 报价历史 | 查看历次报价记录 |
| 最新报价 | 显示最新报价金额 |

### 2.3 商机跟进 [P0]

| 功能 | 说明 |
|:---|:---|
| 添加跟进记录 | 记录跟进方式（电话/面谈）、跟进内容 |
| 跟进附件 | 支持上传附件（会议纪要、通话记录等） |
| 跟进历史 | 查看所有跟进记录（按时间倒序） |
| 修改阶段 | 推进或退回商机阶段 |
| 下次跟进提醒 | 设置下次跟进日期，到期提醒 |

### 2.4 数据统计 [P1]

| 功能 | 说明 |
|:---|:---|
| 商机漏斗图 | 可视化各阶段商机数量/金额 |
| 转化率分析 | 各阶段转化情况 |
| 个人业绩统计 | 按人统计商机数量、金额、转化率 |
| 预期业绩 | 根据阶段加权计算预计签约额 |

- 管理员可看全部

## 五、业务规则

### 5.1 大金额确认

- 设置金额阈值（如：50万）
- 商机金额超过阈值时，状态显示"待确认"
- 需销售经理或管理员确认后才能继续推进

### 5.2 中标转合同/项目

- 商机阶段变为"已签约"时
- 自动触发合同创建流程
- 自动创建项目并关联合同
- 销售填写合同信息，项目经理填写项目信息

### 5.3 阶段流转规则

- 阶段只能向前推进（潜在→已签约）
- 可回退到前一阶段
- 每次阶段变化记录日志
