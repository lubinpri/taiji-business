# 工作看板模块 - 需求规格说明书

## 一、功能清单

### 1.1 首页看板 [P0]

| 功能 | 说明 |
|:---|:---|
| KPI卡片 | 关键指标快速查看 |
| 数据可视化 | 图表展示业务数据 |
| 待办事项 | 待处理业务提醒 |
| 快捷入口 | 快速进入各模块 |

### 1.2 销售看板 [P1]

| 功能 | 说明 |
|:---|:---|
| 商机漏斗 | 各阶段商机数量/金额 |
| 商机统计 | 新增/跟进/转化情况 |
| 销售排行 | 按人/按部门业绩排行 |
| 预期业绩 | 各阶段加权计算 |

### 1.3 项目看板 [P1]

| 功能 | 说明 |
|:---|:---|
| 项目状态分布 | 各状态项目数量 |
| 项目进度 | 进行中项目进度 |
| 风险项目 | 滞后/超期项目预警 |
| 项目排行 | 按利润/按进度排行 |

### 1.4 合同/回款看板 [P1]

| 功能 | 说明 |
|:---|:---|
| 合同金额统计 | 合同总额/执行中/已完成 |
| 回款统计 | 回款金额/回款率 |
| 开票统计 | 开票金额/未开票 |
| 应收款提醒 | 逾期/即将到期 |

### 1.5 成本看板 [P1]

| 功能 | 说明 |
|:---|:---|
| 成本概览 | 总成本/预算对比 |
| 成本趋势 | 月度/季度趋势 |
| 部门成本 | 5个部门成本分布 |
| 利润分析 | 项目/部门利润 |

### 1.6 个人看板 [P2]

| 功能 | 说明 |
|:---|:---|
| 我的商机 | 个人负责的商机 |
| 我的项目 | 个人参与的项目 |
| 我的待办 | 个人待处理事项 |
| 工时汇总 | 个人工时投入 |

## 三、KPI卡片设计

### 3.1 首页KPI

| 指标 | 说明 | 计算方式 |
|:---|:---|:---|
| 商机数量 | 当前有效商机总数 | 状态非"已签约/已失效"的商机 |
| 合同金额 | 合同总额 | 所有合同金额之和 |
| 项目数量 | 在运行项目数 | 状态非"已结项/已终止"的项目 |
| 回款率 | 已回款/合同金额 | 已回款金额÷合同总额×100% |

### 3.2 销售KPI

| 指标 | 说明 |
|:---|:---|
| 商机总数 | 所有商机数量 |
| 预期业绩 | 各阶段金额×阶段转化率之和 |
| 已签约数 | 阶段为"已签约"的商机数 |
| 转化率 | 已签约数÷潜客数×100% |

### 3.3 项目KPI

| 指标 | 说明 |
|:---|:---|
| 项目总数 | 所有项目数量 |
| 风险项目 | 风险等级为"红色"的项目数 |
| 按时完成率 | 按期完成项目÷总项目×100% |
| 平均毛利率 | 项目利润÷项目收入×100% |
