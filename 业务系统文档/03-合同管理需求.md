# 合同管理模块 - 需求规格说明书

## 一、业务流程

```
商机签约 → 合同创建 → 合同执行 → 项目验收 → 成本核算 → 合同归档
              ↓                                    ↓
         合同变更 ←←←←←←←←←←←←←←←←←←←←←←←←← 变更审批
```

### 流程说明

1. **商机签约** → 自动/手动创建合同
2. **合同执行** → 关联多个项目，项目执行中
3. **项目验收** → 项目经理提交项目结项
4. **成本核算** → 商务确认成本结项
5. **合同完成** → 项目结项 + 成本结项 = 合同完成
6. **合同归档** → 合同结束，可查询

## 五、合同完成规则

### 5.1 完成条件（必须同时满足）

```
合同完成 = 所有项目已结项 + 成本已核算
           ↓                    ↓
      项目经理确认        商管人员确认
```

### 5.2 状态流转

```
执行中 → 待验收（所有项目完工）→ 待结项（项目已验收）→ 已完成（成本已核算）
                    ↓
              可终止
```
